Como usar:

1. Coloque o arquivo sped nesta pasta e o renomeie para 'sped_fiscal', mantendo seu formato txt;

2. Clique duas vezes sobre o arquivo 'executar.bat';

3. Feito isso o novo arquivo sera gerado seu nome será assim: 'sped_novo_dd-mm-yyyy_hhmm.txt'.
